Accommodation booking website

Project Overview

As part of an assignment for Yoobee college I was tasked with designing and devolping a single page application that allowed users to search and book accomodation.

The focus of the assignment was devolping my skills in javascript. The process was challenging, but enjoyable. The project is purely hypothetical and is in no way affliated with Tourism New Zealand.

Project Scenario

Tourism has been one of the hardest hit industries by the covid 19 pandemic. As border restrictions loosen up again, TNZ wants to hit the ground running by expanding their services to travellers, making it even easier to explore New Zealand.

TNZ requires a full website build, allowing visitors to find and book accommodation in New Zealand's top holiday spots.

Solution

The proposed solution is a single page web application that allows users to input information and find appropriate accommodation based on their individual requirements.

After selecting accommodation, the site will then allow users to pre-order their breakfast meals. The website will be devolped in such a way that allows for growth as more vendors join the site.
